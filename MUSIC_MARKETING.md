## 1. Playlist Submission Strategy

### Spotify Editorial Pitching
- Submit via **Spotify for Artists** at least **7 days before release** (unreleased tracks only)
- Choose the single most relevant genre / mood / context
- Write a concise pitch (≤ 500 chars): describe the sound, the story, the target listener — not yourself
- One pitch per release; you cannot re-pitch the same track

### Independent Playlist Curators
- Target curators via **SubmitHub** — use credits, not free slots, for higher response rates
- Filter by genre match (electronic / DJ / ambient / etc.)
- Minimum 7–14 days lead time
- Track all submissions in `RELEASE_CALENDAR.md`

### Algorithmic Playlists (Spotify Discover Weekly / Release Radar)
- Upload via DistroKid at least 3 weeks before release so Spotify can index correctly
- First week streaming velocity matters — coordinate release day push (social + email)
- Encourage saves, not just streams (saves signal intent to Spotify's algorithm)

### YouTube Music / Apple Music
- Apple Music editorial pitch: use **Apple Music for Artists** (no self-serve editorial pitch; focus on DSP profile quality)
- YouTube Music: ensure auto-generated topic channel is claimed; upload official audio/video to main channel

## 2. Social Media Release Plan Template

### T-28 days (4 weeks out)
- [ ] Announce release date + title on all channels
- [ ] Share cover art reveal
- [ ] Post 15–30 s teaser clip (hook section)

### T-14 days (2 weeks out)
- [ ] Behind-the-scenes: studio / production process snippet
- [ ] Pre-save / pre-add link live (DistroKid HyperFollow)
- [ ] Begin curator outreach on SubmitHub

### T-7 days (1 week out)
- [ ] Second teaser (different section of track)
- [ ] Reminder post with pre-save link
- [ ] Spotify for Artists editorial pitch submitted (if not already done)

### T-1 day (day before release)
- [ ] "Tomorrow it drops" countdown post
- [ ] Stories / Reels with track preview

### Release day (T-0)
- [ ] Full release announcement post with streaming links
- [ ] Pin post/story with links to all platforms
- [ ] Engage with every comment for the first 2 hours (algorithm boost)
- [ ] Share to relevant communities (Reddit, Discord, Facebook groups)

### T+7 days (1 week after release)
- [ ] Share streaming milestone if notable (first 1K / 5K streams)
- [ ] Lyric card or quote graphic from the track
- [ ] Thank-you post to community

### T+30 days (1 month after release)
- [ ] Review streaming analytics — top playlists, countries, listener demographics
- [ ] Share any press or playlist placements earned
- [ ] Begin teasing next release

## 3. Platform-Specific Optimisation

### Spotify
- **Artist profile**: bio, header image, social links, artist pick updated each release
- **Canvas**: 3–8 s looping vertical video (1080 × 1920) — upload via Spotify for Artists
- **Clips**: 30 s short-form clip for mobile listeners
- **Playlist pitching**: single editorial pitch per unreleased track
- **Lyrics**: upload LRC/plain-text lyrics via DistroKid for time-synced display
- **Spotify Codes**: generate and use in print/digital promo

### Apple Music
- **Artist page**: update bio and photos via Apple Music for Artists
- **Spatial Audio / Dolby Atmos**: consider Atmos mix for catalogue differentiation
- **iTunes Store**: price single at NZD 2.19 / USD 1.29 (standard) or offer free
- **Music Videos**: upload MV through DistroKid for Apple Music video delivery

### YouTube Music
- **Official Artist Channel**: claim via YouTube for Artists (requires 3+ releases)
- **Official Audio video**: upload to YouTube channel simultaneously with release; YouTube Music auto-populates from channel if linked
- **Shorts**: post 60 s vertical clip on release day for algorithm exposure
- **Description**: include all streaming links, timestamped if album; add lyrics in description as plain text

### TikTok / Instagram Reels / Facebook Reels
- 15 s hook clip auto-selected by platform from DistroKid delivery
- Create original content using the track — tutorials, behind-the-scenes, duet-bait prompts
- Use trending sounds + original sound strategy together

### Bandcamp
- Upload manually (Bandcamp not in DistroKid delivery)
- Set price: NZD 2.00 minimum / name-your-price for early releases
- Tag genres accurately — Bandcamp discovery is tag-driven
- Enable "send to Bandcamp" for fans via HyperFollow link
- Bandcamp Friday (first Friday of each month) — release or discount timed to this

## 4. Press & Blog Outreach

- Identify 10–15 electronic music blogs / Aotearoa NZ music media (Flying Nun doesn't count — aim at digital-first outlets)
- Send one-paragraph pitch + streaming link (Spotify / SoundCloud unlisted) + cover art + bio
- Lead time: 2–3 weeks before release
- Do not send mass-blast emails — personalise each pitch

### NZ-Specific Targets
- **Rip It Up / Audioculture** — NZ music history + new releases
- **The Wireless (RNZ)** — digital-first NZ music coverage
- **NZ Musician** — industry trades, gear, production focus
- **Flicks / Stuff Entertainment** — broader NZ arts/culture for crossover
- **SoundSpheremag** (international) — electronic/ambient focus, accepts NZ acts

---

## 5. Email / Community List

- Build a direct fan list via SongPal or a simple Mailchimp / Resend flow
- Release-day email: short, personal, one clear CTA (stream link)
- Do not over-email — maximum one per release cycle
