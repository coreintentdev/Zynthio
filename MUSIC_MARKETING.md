## Release Strategy Overview

### Recommended Release Cadence
- **Singles:** Every 4–8 weeks to maintain algorithmic momentum
- **EP:** After 3–4 singles have performed well
- **Album:** After 6–12 months of consistent singles

### Release Timeline (Per Track)
```
T-28 days  → Finalise master + artwork
T-21 days  → Upload to DistroKid, submit Spotify editorial pitch
T-14 days  → Announce pre-save, start teaser campaign
T-7  days  → Ramp up socials — audio snippets, BTS content
T-3  days  → Final reminder push, presave push
T-1  day   → "Tomorrow" reminder post
T-0  (Release Day) → Launch posts across all platforms
T+7  days  → Share streaming links + milestone update
T+14 days  → Playlist pitch follow-ups, analytics review
T+30 days  → Performance report, decide next release
```

## Playlist Submission Strategy

### Spotify Editorial (Official)
- Submit via **Spotify for Artists** dashboard at least **7 days before release**
- Only available on unreleased tracks (pre-release)
- Fill in ALL fields: mood, instrumentation, style
- Be specific: "dark, driving bassline, 128 BPM, euphoric drop, made for late-night driving"
- Target playlists: list genre-appropriate editorial playlists to reference

### Independent Playlist Curators
Submit to independent curators via:
- **SubmitHub** (https://www.submithub.com) — paid credits, high ROI for Electronic
- **Groover** (https://groover.co) — curator + blog pitching
- **Playlist Push** — automated curator network

**Curator Pitch Template:**
```
Subject: [Track Title] by DJ Zynrose — [Genre] — New Release [Date]

Hi [Curator Name],

I'm DJ Zynrose, an electronic music producer from New Zealand.
My new track "[Track Title]" drops [Release Date] on [Platforms].

[2-sentence description of the track: mood, energy, instruments, influences]

I think it would fit well in your [Playlist Name] playlist because [specific reason].

DistroKid link / Pre-save: [URL]
Soundcloud preview: [URL]

Thanks for your time,
Corey McIvor / DJ Zynrose
corey@coreyai.ai
```

## Platform-Specific Optimisation

### Spotify
- [ ] Claim Spotify for Artists profile
- [ ] Upload artist photo (min 750×750 px)
- [ ] Write artist bio (max 1500 chars — focus on story, not credits)
- [ ] Set hometown: New Zealand
- [ ] Pin latest release on profile
- [ ] Create an artist playlist featuring own tracks + influences
- [ ] Submit to editorial every release (T-21 days)
- [ ] Enable Spotify Canvas (vertical looping video, 3–8 sec, 9:16 ratio)
- [ ] Spotify Clips (short form, similar to Reels)
- [ ] Optimise track metadata: mood + style tags in Spotify for Artists

**Spotify Canvas Specs:**
- Format: MP4 or JPEG sequence
- Aspect ratio: 9:16 (vertical)
- Length: 3–8 seconds, loops seamlessly
- Resolution: 720×1280 minimum
- No text overlays, no explicit content

### Apple Music
- [ ] Claim Apple Music for Artists profile
- [ ] Upload artist photo
- [ ] Write artist bio
- [ ] Create Apple Music playlist (curated by DJ Zynrose)
- [ ] Submit to Apple Music editorial via Apple Music for Artists → "Upcoming Releases"
- [ ] Add animated cover art (Motion artwork — optional but premium feel)

**Apple Music Motion Artwork Specs:**
- Format: MOV, H.264
- Dimensions: 3000×3000 px
- Duration: up to 30 seconds, loops

### YouTube Music
- [ ] Claim YouTube Official Artist Channel (OAC)
- [ ] Upload visualiser or lyric video for each release
- [ ] Premiere feature — schedule video premiere for release day
- [ ] Add to YouTube Music playlists
- [ ] Enable Content ID via DistroKid to monetise third-party uses

**YouTube Video Specs (Visualiser/Lyric Video):**
- Resolution: 3840×2160 (4K) or 1920×1080 minimum
- Format: MP4, H.264
- Audio: 320 kbps AAC or original WAV muxed in
- Thumbnail: 1280×720 px, JPG

### TikTok / Instagram Reels / YouTube Shorts
- [ ] Enable TikTok distribution via DistroKid
- [ ] Create 15–30 second hook clip from best part of track
- [ ] Post as original audio — encourage duets/stitches
- [ ] Trending audio format: start with hook, show production process
- [ ] Post 3–5 TikToks per release week

**Short-Form Video Hook Ideas:**
```
- "I made this in [X hours] — here's the drop"
- "New track out [date] — this is the section I worked on longest"
- "BTS: building the bassline for [Track Title]"
- "POV: you're at [location] at midnight" + atmospheric clip
- Studio session timelapse with track audio
```

### Beatport / Traxsource (if Electronic/Dance)
- [ ] Enable Beatport via DistroKid (requires Electronic genre)
- [ ] Submit to Beatport editorial charts via their artist program
- [ ] Traxsource: submit via DistroKid's Traxsource toggle
- [ ] DJ promotional copies: send WAVs to DJs via Soundcloud Pro or Promo.DJ

## Social Media Release Plan Template

Copy and adapt this for each release. Replace all `[brackets]`.

### Pre-Release Week 3 (T-21 days)
**Platform: Instagram / TikTok / X (Twitter)**
```
Post 1 — Teaser (image or short clip)
Caption: "Something's coming. [Release Date]. 🎵"
Hashtags: #DJZynrose #SongPal #ComingSoon #ElectronicMusic #NewMusic
```

### Pre-Release Week 2 (T-14 days)
**Platform: All**
```
Post 2 — Pre-save announcement
Caption: "New track '[Title]' dropping [Date].
Pre-save now → [LINK]
[2-sentence description of the vibe]"

Story: Countdown sticker set to release date
```

### Pre-Release Week 1 (T-7 days)
```
Post 3 — Audio teaser (15–30 sec clip)
Caption: "One week. '[Title]' drops [Date].
Pre-save: [LINK]"

Post 4 — BTS / production content
Caption: "Here's how I built the [element] in '[Title]'..."
Format: Reel/TikTok showing DAW, process, or performance
```

### Release Day (T-0)
```
Post 5 — Launch post (cover art + streaming link)
Caption:
"🚀 OUT NOW — '[Title]' by DJ Zynrose
Stream on Spotify, Apple Music, YouTube Music + everywhere else.
→ [SmartLink / Linktree URL]

[2–3 sentences about the track — the story, the feel, what inspired it]

#DJZynrose #SongPal #NewRelease #ElectronicMusic #[GenreTag] #NewZealand"

Story: Share streaming link, add poll ("Have you heard it yet?")
TikTok: "It's out. Link in bio. 🎵"
```

### Post-Release Week 1 (T+7 days)
```
Post 6 — Milestone / gratitude
Caption: "One week since '[Title]' dropped. [streams/saves/plays if shareable].
Thank you. Next track is already in the works. 🎵"

Post 7 — Engagement driver
Caption: "What part of '[Title]' hit you hardest? Drop a timestamp 👇"
```
