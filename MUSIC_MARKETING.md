## 1. Playlist Submission Strategy

### Priority Tier 1 — Editorial (Platform-Direct)

Submit these **at least 7 days before release date** via the platform's own tools.

| Platform | Submission Tool | Notes |
|----------|----------------|-------|
| Spotify | Spotify for Artists → Upcoming → Pitch a Song | One song per release only; pitch your strongest cut |
| Apple Music | Apple Music for Artists → Submit for Editorial Consideration | Less structured than Spotify but worth doing |
| Amazon Music | Amazon Music for Artists → Pitch | Growing platform; lower competition |
| Deezer | editors@deezer.com or via distributor pitch tool | Email pitch accepted |

**Pitch template (Spotify for Artists):**
```
Genre: [Primary genre]
Mood: [e.g. energetic / melancholic / euphoric]
Style: [Sounds like: X meets Y]
Instrumentation: [e.g. synthesizers, 808 drums, vocals]
Story: [1–2 sentences about the track's concept]
```

---

### Priority Tier 2 — Independent Playlist Curators

Submit to curators 5–10 days before release. Use these tools:

| Tool | Notes |
|------|-------|
| **SubmitHub** | Paid credits per submission; curators respond within 48h; best ROI for indie |
| **Groover** | European-focused; good for electronic / DJ music |
| **Playlist Push** | Organic playlist placement; Spotify-focused |
| **Soundplate** | Free curator submissions; lower hit rate but no cost |
| **PlaylistSupply** | Aggregates curator contact info |

**Curator outreach email template:**
```
Subject: [PLAYLIST NAME] submission — DJ Zynrose — "[Track Title]"

Hi [Curator Name],

I'm DJ Zynrose, a New Zealand-based electronic music producer. I'd love to 
submit "[Track Title]" for consideration on [Playlist Name].

The track is [genre] — [one-line description]. It releases on [date] via SongPal / DistroKid.

Streaming link: [Spotify / SoundCloud link]
DistroKid HyperFollow: [link]

Thanks for your time and for supporting independent artists.

Corey McIvor / DJ Zynrose
```

## 3. Platform-Specific Optimisation

### Spotify

| Action | Detail |
|--------|--------|
| Artist bio | Keep updated; 1–2 paragraphs; include NZ origin story |
| Artist pick | Pin latest release to top of artist page |
| Canvas | Upload 8-sec looping video (vertical, no audio) — increases saves by ~20% |
| Storyline | Add behind-the-scenes text/image to albums |
| Playlist placement | Aim for Release Radar + Discover Weekly via algorithmic triggers (saves & listens-to-completion) |
| Algorithmic hack | First 30 seconds must hook — listeners who skip early kill the algorithm |

**Key Spotify metrics to track:**
- Streams
- Saves (saves-to-streams ratio is the key metric)
- Playlist adds (editorial vs. algorithmic vs. listener)
- Listener countries (identify growth markets)
- Listener-to-follower conversion

---

### Apple Music

| Action | Detail |
|--------|--------|
| Artist profile | Complete all fields — photo, bio, links |
| Shazam | Apple owns Shazam; ensure metadata is correct for Shazam recognition |
| iTunes artwork | Same 3000×3000 standard — also appears on Apple Watch |
| Spatial Audio | If releasing on Apple Music, consider Dolby Atmos mix for Spatial Audio badge |
| Radio | Submit to Beats 1 / Apple Music Radio shows for feature consideration |

---

### YouTube Music

| Action | Detail |
|--------|--------|
| OAC (Official Artist Channel) | Claim via DistroKid or YouTube directly |
| YouTube Content ID | Enabled via DistroKid — monetises user videos using your music |
| Shorts | Upload vertical clips; use your own audio to claim shorts usage |
| Community posts | Use YouTube Community tab on release days |
| Metadata | Match title/artist exactly to what's on other platforms |

---

### TikTok / Instagram Reels / YouTube Shorts

| Action | Detail |
|--------|--------|
| Hook | First 3 seconds must grab attention — no intro fade-ins |
| Audio clip selection | Choose the catchiest 15–30 sec section for DistroKid's TikTok clip |
| Sound page | Once uploaded, find your sound page on TikTok and engage with all videos using it |
| Duets / Stitches | Enable for user participation and organic spread |
| Hashtag strategy | 3–5 niche hashtags + 1–2 broad (#NewMusic, #ElectronicMusic) |

---

### Beatport / Traxsource (DJ-Specific)

| Action | Detail |
|--------|--------|
| Beatport | Submit directly or via a Beatport-connected distributor (not all DK plans include it) |
| Traxsource | House / deep house focus — if genre-relevant |
| Juno Download | UK-based; good for drum & bass, techno, house |
| DJ charts | Getting onto a prominent DJ's chart on Beatport is major social proof |
