# Artwork Directory — DJ Zynrose / SongPal

Store cover art files here.

## File Naming Convention

```
TRK-XXX_[Track-Title]_COVER.jpg
TRK-XXX_[Track-Title]_COVER.png
```

## Specs (DistroKid / DSP Requirements)

| Spec | Requirement |
|------|------------|
| Dimensions | 3000 × 3000 px (square, mandatory) |
| Format | JPG or PNG |
| Colour space | RGB |
| Resolution | 72 dpi minimum (300 dpi recommended) |
| Max file size | 10 MB (keep under 5 MB where possible) |

## Restrictions

- No URLs, social handles, pricing, or store logos in artwork
- No blurriness or visible compression artefacts
- Explicit imagery only if release is flagged explicit in DistroKid
