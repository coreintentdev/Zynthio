# Lyrics Directory

This directory contains lyric files for all DJ Zynrose tracks.

## File Naming Convention

```
TrackTitle_DJZynrose_LYRICS.txt
```

## File Format

Each lyrics file should follow this structure:

```
TITLE: [Track Title]
ARTIST: DJ Zynrose
ALBUM: [Album / Single]
STATUS: draft | final
LEGAL: original | sample-cleared | instrumental

---

[Verse 1]
Lyrics here...

[Chorus]

[Chorus]

[Bridge]
Lyrics here...

[Outro]
Lyrics here...
```
