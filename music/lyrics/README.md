## File Naming Convention

```
[track-slug].txt        Plain text lyrics (for DistroKid upload)
[track-slug].lrc        Timed LRC format (for synced lyrics on Spotify/Apple Music)
```

**Track slug format:** lowercase, hyphens instead of spaces, no special characters  
Example: `neon-sunrise.txt`, `neon-sunrise.lrc`

---

## Timed LRC Format (.lrc)

```
[ti:Track Name]
[ar:DJ Zynrose]
[al:Album Title]
[au:Corey McIvor]
[length:3:45]

[00:00.00] (intro)
[00:12.50] First lyric line here
[00:15.80] Second lyric line here
...
```

LRC timestamps use format `[MM:SS.mm]` — minutes:seconds.milliseconds
