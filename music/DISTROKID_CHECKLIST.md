## Per-Track Requirements

Copy this checklist for each track before submitting to DistroKid.

---

### [ ] Track Title
- Must match exactly across all platforms (Spotify, Apple Music, YouTube Music)
- No ALL CAPS unless stylistic intent
- No trailing punctuation

### [ ] Artist Name
- Primary: `DJ Zynrose`
- Featuring artists: list as `ft. [Name]` in title or in the featuring field — not both

### [ ] Audio File
- [ ] Format: **WAV** (preferred) or FLAC
- [ ] Bit depth: **16-bit minimum**, 24-bit preferred
- [ ] Sample rate: **44.1 kHz minimum**, 48 kHz preferred
- [ ] No clipping — peaks below -0.1 dBFS
- [ ] Lufs target: **-14 LUFS integrated** (Spotify standard)
- [ ] File named: `TrackTitle_DJZynrose_MASTER.wav`
- [ ] Silence at start: < 1 second
- [ ] Silence at end: 1–2 seconds

### [ ] Cover Art
- [ ] Dimensions: **3000 × 3000 pixels**
- [ ] Format: **JPG or PNG**
- [ ] Colour space: **RGB** (not CMYK)
- [ ] Resolution: **72 DPI minimum** (300 DPI recommended)
- [ ] File size: under 10 MB
- [ ] No URLs, pricing, or social handles in the image
- [ ] No copyrighted logos or imagery without clearance
- [ ] File named: `TrackTitle_DJZynrose_COVER.jpg`

### [ ] Metadata
- [ ] **Title** — confirmed spelling
- [ ] **Primary Artist** — `DJ Zynrose`
- [ ] **Featuring Artist(s)** — listed if applicable
- [ ] **Album / Single name** — matches cover art text exactly
- [ ] **Genre** — primary + secondary (DistroKid allows 2)
- [ ] **Release Date** — set minimum 3–5 business days out (Spotify pitching requires 7 days)
- [ ] **Language** — English (or correct language)
- [ ] **Explicit** — flagged yes/no accurately
- [ ] **ISRC** — DistroKid auto-assigns; note it after upload for your records
- [ ] **UPC** — DistroKid auto-assigns for albums; note it

### [ ] Lyrics
- [ ] Uploaded to DistroKid lyrics field (plain text, time-stamped preferred)
- [ ] Lyrics file saved in `/music/lyrics/` in this repo
- [ ] Marked: original / sample-cleared / instrumental

### [ ] Legal Clearance
- [ ] Fully original composition — no samples
- [ ] OR — samples licensed and documented in `/music/sample-clearances/`
- [ ] Songwriter credits confirmed
- [ ] Publisher: self-published (or registered publisher name)

### [ ] Pre-Release Tasks
- [ ] Spotify for Artists pre-save link set up (DistroKid HyperFollow or Hypeddit)
- [ ] Apple Music pitched via DistroKid (requires 7+ days lead time)
- [ ] Spotify editorial pitch submitted (requires 7+ days, done via Spotify for Artists dashboard)
- [ ] Social media announcement scheduled (see MUSIC_MARKETING.md)

### [ ] Post-Release Verification
- [ ] Track live on Spotify — ID confirmed
- [ ] Track live on Apple Music — ID confirmed
- [ ] Track live on YouTube Music
- [ ] ISRC recorded in TRACK_MANIFEST.md
- [ ] DistroKid earnings dashboard checked at 30 days
