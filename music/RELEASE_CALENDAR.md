## How to Use This Calendar

### Release Cadence Recommendation
- **Singles:** one every 4–8 weeks keeps algorithm momentum
- **EPs:** 1–2 per year (4–6 tracks each)
- **Albums:** 1 per year if volume warrants
- **Friday releases:** preferred (Spotify New Music Friday; Apple Music algorithm)

## Post-Release Follow-Up

Complete within 30 days of release:

- [ ] Verify all store listings are live and correct
- [ ] ISRC confirmed on Spotify for Artists
- [ ] Track added to own Spotify playlists
- [ ] Streaming numbers logged (first-week snapshot)
- [ ] Social media engagement recap saved
- [ ] Any press/blog coverage saved to `/music/press/` folder
- [ ] `TRACK_MANIFEST.md` status updated to `released`
- [ ] Royalty split confirmed (if collaborators involved)
- [ ] Lessons learned noted for next release
