> **Jurisdiction:** New Zealand
> **Last updated:** 2026-04-16

---

## Purpose

This directory stores plain-text lyric files for all DJ Zynrose tracks.

- One `.txt` file per track
- File naming convention: `track-name-slug.txt` (lowercase, hyphens, no spaces)
- Lyrics are **original content only** — never reproduce copyrighted material

---

## File Naming Convention

```
artist-track-title-version.txt

Examples:
  dj-zynrose-track-title.txt
  dj-zynrose-track-title-clean.txt
  dj-zynrose-track-title-feat-artistname.txt
```

---

## Lyrics File Format

Each lyrics file should follow this structure:

```
TITLE: [Track Title]
ARTIST: DJ Zynrose
ALBUM: [Album / EP / Single title]
STATUS: draft | ready | released
DATE: YYYY-MM-DD

---

...
```

## Status Labels

| Status | Meaning |
|--------|---------|
| `draft` | Lyrics written but not finalised |
| `ready` | Finalised, approved for upload to DistroKid |
| `released` | Live on streaming platforms |

## Usage

1. Create a new `.txt` file following the naming convention above
2. Write lyrics using the format template above
3. Set `STATUS:` to `draft`
4. When finalised, update status to `ready` and upload to DistroKid's lyrics field
5. After release, update status to `released` and update the table above
6. Update `TRACK_MANIFEST.md` to mark lyrics as complete for the track
