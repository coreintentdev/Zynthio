# Lyrics Directory — DJ Zynrose / SongPal

Store one plain-text lyrics file per track here.

## File Naming Convention

```
TRK-XXX_[Track-Title].txt
```

Example: `TRK-001_Signal-Lost.txt`

## Format Requirements

- Encoding: UTF-8, no BOM
- Line endings: LF (Unix)
- Sections labelled: `[Verse 1]`, `[Chorus]`, `[Bridge]`, `[Outro]`, etc.
- Timestamps (LRC format) added when available for sync-upload to Spotify/Apple Music:
  ```
  [00:15.20] First line of verse one
  [00:18.50] Second line continues here
  ```
