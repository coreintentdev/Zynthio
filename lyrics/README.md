This directory holds lyric files for all DJ Zynrose / Corey McIvor tracks.

**All content in this directory is original, copyright Corey McIvor.**  
**No third-party or copyrighted lyrics are stored here.**

---

## File Naming Convention

```
[track-slug].md          — plain lyrics (for reference, Musixmatch, DistroKid)
[track-slug].lrc         — timed lyrics (for Apple Music / Spotify sync, optional)
```

### Track slug format
- Lowercase
- Hyphens for spaces
- No special characters
- Example: `midnight-frequency.md`, `signal-break.lrc`

---

## Lyric File Template

Create a new `.md` file per track. Copy this template:

```markdown

**Artist:** DJ Zynrose  
**Written by:** Corey McIvor  
**Copyright:** © [YEAR] Corey McIvor. All rights reserved.  
**Status:** draft / ready / released  
**ISRC:** [NZ-XXX-YY-NNNNN or pending]

---

[lyrics here]

[lyrics here]

[Verse 2]

[Bridge]

[Outro]

---

*Original composition. No samples. NZ jurisdiction.*
```

## Instrumental Tracks

For instrumental tracks, create a file with:

```markdown
# [Track Title]
**Artist:** DJ Zynrose  
**Written by:** Corey McIvor  
**Type:** Instrumental  
**Status:** [status]
```
