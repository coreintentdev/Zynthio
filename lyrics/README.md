## Purpose

This directory contains lyrics files for all DJ Zynrose tracks.  
Lyrics are used for:
- DistroKid upload (enables Spotify lyrics display)
- Musixmatch / LyricFind sync
- Copyright registration with APRA AMCOS
- Press and promotional materials

## File Naming Convention

```
lyrics/<track-name-kebab-case>.md
```

Examples:
- `lyrics/into-the-grid.md`
- `lyrics/neon-cascade.md`
- `lyrics/signal-lost.md`
