## Purpose

This directory holds plain-text lyric files for all DJ Zynrose tracks.

All lyrics in this directory are **original compositions** by Corey McIvor.
No copyrighted third-party content is reproduced here.

---

## File Naming Convention

```
[TrackID]_[TrackTitle_underscored].txt
```

Examples:
- `001_Neon_Skyline.txt`
- `002_Glass_Box.txt`

---

## File Format

Each lyric file should follow this structure:

```
TRACK: [Track Title]
ARTIST: DJ Zynrose
WRITER: Corey McIvor
STATUS: draft / final
DATE: YYYY-MM-DD

---

[VERSE 1]
Lyrics here...

[CHORUS]
Lyrics here...

[VERSE 2]
Lyrics here...

[BRIDGE]
Lyrics here...

[OUTRO]
Lyrics here...
```

---

## LRC (Synced Lyrics) Files

For Spotify Canvas sync and Apple Music timed lyrics, create `.lrc` files:

```
[TrackID]_[TrackTitle_underscored].lrc
```

LRC format:
```
[00:00.00] First line of lyrics
[00:05.30] Second line
[00:10.15] Chorus starts
```
