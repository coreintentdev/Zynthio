## Universal Requirements (Every Release)

### Audio File
- [ ] WAV format (required — DistroKid transcodes for all platforms)
- [ ] Sample rate: 44.1 kHz or 48 kHz
- [ ] Bit depth: 16-bit minimum, 24-bit preferred
- [ ] No clipping — peaks below 0 dBFS (target: -1 dBFS true peak)
- [ ] Loudness: -14 LUFS integrated (Spotify standard) or louder for electronic
- [ ] No silence at start longer than 2 seconds
- [ ] No silence at end longer than 5 seconds
- [ ] File naming: `<TrackTitle>_master.wav` (no spaces, use underscores)

### Cover Art
- [ ] Minimum size: 3000 x 3000 pixels
- [ ] Format: JPG or PNG (PNG preferred for quality)
- [ ] Color mode: RGB (not CMYK)
- [ ] Resolution: 72 DPI minimum (300 DPI recommended)
- [ ] No blurriness or pixelation
- [ ] No URLs, social handles, or website addresses in artwork
- [ ] No explicit imagery (platform-dependent)
- [ ] No misleading artist/title text that doesn't match metadata
- [ ] File size: under 10 MB
- [ ] File naming: `<ReleaseName>_cover.png`

## Platform-Specific Notes

### Spotify
- Lyrics uploaded via DistroKid appear on Spotify natively
- Pitch to editorial playlists via **Spotify for Artists** — submit at least 7 days before release
- Canvas videos (3–8 second looping video) can be added post-release via Spotify for Artists

### Apple Music
- Artwork must be 3000x3000 — no exceptions
- Dolby Atmos / Spatial Audio: upload separately if available (24-bit WAV + ADM BWF)

### YouTube Music
- DistroKid auto-delivers to YouTube Music
- YouTube Content ID is enabled via DistroKid's optional "YouTube Money" add-on
- Separate lyric videos should be uploaded manually to your YouTube channel

### TikTok / Instagram Reels / Facebook
- DistroKid delivers a clip (usually 60s) to TikTok automatically
- Ensure the "hook" of your track is within the first 60 seconds

### Beatport / Traxsource (DJ Pools)
- DistroKid does NOT deliver to Beatport/Traxsource — these require separate distribution
- Consider **Label Worx**, **FUGA**, or direct label applications for these platforms
