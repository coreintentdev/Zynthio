## Per-Release Checklist

Use one copy of this checklist per single, EP, or album submission.

---

### 1. AUDIO FILE

- [ ] Format: **WAV** (preferred) or FLAC — no MP3 masters
- [ ] Sample rate: **44.1 kHz** minimum (96 kHz accepted)
- [ ] Bit depth: **16-bit minimum**, 24-bit preferred
- [ ] Channels: **Stereo** (not mono, not 5.1)
- [ ] Loudness: mastered to **-14 LUFS integrated** (Spotify standard)
- [ ] True peak: **≤ -1 dBTP**
- [ ] No clipping, no DC offset, no silence errors at head/tail
- [ ] File named: `[ArtistSlug]_[TrackSlug]_MASTER.wav`

---

### 2. COVER ART

- [ ] Dimensions: **3000 × 3000 px** (square)
- [ ] Format: **JPG or PNG**
- [ ] Resolution: **72 DPI minimum** (300 DPI recommended for print)
- [ ] Color mode: **RGB** (not CMYK)
- [ ] File size: < 10 MB
- [ ] No explicit text unless Explicit flag is set
- [ ] No URLs, social handles, or pricing in artwork
- [ ] No third-party logos or trademarks without clearance
- [ ] File named: `[ArtistSlug]_[TrackSlug]_COVER.jpg`

---

### 3. METADATA

- [ ] **Title** — exact title, correct capitalisation, no trailing spaces
- [ ] **Artist name** — `DJ Zynrose` (consistent across all releases)
- [ ] **Primary genre** — e.g. Electronic
- [ ] **Secondary genre** — e.g. House / Lo-fi / Ambient
- [ ] **Release date** — at least 7 days in advance for standard; 3 weeks for pitching
- [ ] **Record label** — `SongPal` (or `Self-Released` if not yet branded)
- [ ] **Language** — English (or Instrumental)
- [ ] **Explicit content** — flagged if applicable
- [ ] **UPC/EAN** — auto-generated by DistroKid or manually entered
- [ ] **ISRC** — one per track; register via Recorded Music NZ if not auto-assigned

---

### 4. LYRICS

- [ ] Lyrics file created at `lyrics/[slug].md`
- [ ] Lyrics are original (no third-party copyright)
- [ ] Timed lyrics (LRC format) created if submitting to Apple Music / Spotify Canvas
- [ ] No uncleared samples referenced in lyrics
- [ ] Mark `[Instrumental]` if no vocals
