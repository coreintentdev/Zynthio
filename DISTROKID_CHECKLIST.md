## Per-Track Upload Checklist

Use one copy of this checklist per release. Mark each item `[x]` when complete.

### 1. Audio File

- [ ] Format: WAV (preferred) or FLAC — 16-bit or 24-bit, 44.1 kHz minimum
- [ ] No clipping: peak level ≤ −0.3 dBFS
- [ ] Loudness target: −14 LUFS integrated (Spotify standard)
- [ ] Silence at head: ≤ 0.5 s
- [ ] Silence at tail: ≤ 2 s (unless intentional fade)
- [ ] File naming: `TRK-XXX_DJ-Zynrose_[Track-Title]_MASTER.wav`

### 2. Cover Art

- [ ] Dimensions: 3000 × 3000 px (square, mandatory)
- [ ] Format: JPG or PNG
- [ ] Colour space: RGB (not CMYK)
- [ ] Resolution: 72 dpi minimum (300 dpi recommended for print use)
- [ ] No blurriness, pixelation, or compression artefacts
- [ ] No URLs, social handles, or pricing in the artwork
- [ ] No explicit imagery unless release is marked explicit
- [ ] File naming: `TRK-XXX_DJ-Zynrose_[Track-Title]_COVER.jpg`

### 3. Metadata

- [ ] Track title (exactly as it should appear on stores)
- [ ] Primary artist: `DJ Zynrose`
- [ ] Featuring artists (if any) listed correctly
- [ ] Album / EP / Single title
- [ ] Genre (primary + secondary)
- [ ] Subgenre
- [ ] Release date (YYYY-MM-DD, ≥ 7 days ahead for Spotify pitch eligibility; ≥ 3 weeks recommended)
- [ ] Language of lyrics
- [ ] Explicit content flag: yes / no / clean version available
- [ ] Composer / songwriter credits (legal name: Corey McIvor)
- [ ] Producer credit: DJ Zynrose
- [ ] Copyright year and owner (e.g. `℗ 2026 Corey McIvor`)
- [ ] Record label: SongPal (or leave blank for self-release)
- [ ] UPC: (auto-assigned by DistroKid or supply own)
- [ ] ISRC: (auto-assigned by DistroKid or supply own)

### 4. Lyrics

- [ ] Lyrics file prepared (`lyrics/TRK-XXX_[Track-Title].txt`)
- [ ] Lyrics uploaded to DistroKid (enables time-synced lyrics on Apple Music / Spotify)
- [ ] No copyrighted third-party lyrics included

### 5. Stores & Distribution

- [ ] Spotify selected
- [ ] Apple Music / iTunes selected
- [ ] YouTube Music selected
- [ ] Amazon Music selected
- [ ] Tidal selected
- [ ] Deezer selected
- [ ] TikTok / Instagram / Facebook (social platforms) — confirm opt-in
- [ ] Bandcamp (manual upload — not via DistroKid; set NZD pricing)
- [ ] Territory: Worldwide (default) unless geo-restricted

### 6. Splits & Royalties

- [ ] Collaborator splits set in DistroKid (if applicable)
- [ ] Publishing / songwriter share allocated correctly
- [ ] Bank/payment details current in DistroKid account

### 7. Pre-Release

- [ ] Spotify for Artists pitch submitted (≥ 7 days before release)
- [ ] Apple Music for Artists connect confirmed
- [ ] Pre-save link created (optional: DistroKid HyperFollow or Submithub)

## Blocking Items Quick-Reference

If any of the following are missing, the upload **cannot proceed**:

1. Stereo WAV master file
2. 3000 × 3000 cover art (JPG/PNG)
3. Track title and primary artist name
4. Release date
